# Tools

## LindaIDE
A development tool for LINDA smart contracts. It provides a visual interface that supports compiling, debugging, and running Solidity smart contracts.
[https://www.lindaide.io/](https://www.lindaide.io/)

## LindaBox
A deployment tool for LINDA smart contracts. It supports the compilation, deployment, and migration of Solidity smart contracts.
[https://lindabox.io/#/](https://lindabox.io/#/)

## LindaWeb
A Javascript library for developing LINDA smart contracts. It provides interfaces for interacting with the LINDA Mainnet, as well as for deploying and calling contracts.
[https://lindaweb.network/](https://lindaweb.network/)

## LindaGrid
An event query service for LINDA smart contracts. It allows you to query event log information written in smart contracts.
[https://www.lindagrid.io/](https://www.lindagrid.io/)

## Trident
A lightweight Java SDK. It includes a series of libraries to help developers easily work with the LINDA network's system and smart contracts.
[https://lindaprotocol.github.io/trident/](https://lindaprotocol.github.io/trident/)
